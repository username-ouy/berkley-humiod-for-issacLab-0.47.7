### Installation

- Install Isaac Lab, see
  the [installation guide](https://isaac-sim.github.io/IsaacLab/main/source/setup/installation/pip_installation.html). **Please use
  [IsaacLab 0.47.7 with IsaacSim 5.1.0]**.

- Using a python interpreter that has Isaac sLab installed, install the library

```
cd skyentific_poclegs
python -m pip install -e .
```

### Use
- train and play
```
cd berkley-humiod-for-issacLab-0.47.7
sh run_train.sh
```
